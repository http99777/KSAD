# KSAD :bomb 
В рамках курса предусмотрено 17 лекционных занятий, где подробно
разбираются основные концепции и подходы, а также 34 лабораторных занятия, на
которых вы сможете применить полученные знания на практике. Для закрепления
материала – 12 лабораторных работ по PostgreSQL, 6 контестов на алгоритмы и
14 задач на структуры данных.
